# Модуль 2. Реализованные улучшения

- адаптивная верстка под мобильный экран `390x844`;
- inline-подсказки и сообщения об ошибках рядом с полями форм;
- поле даты заявки в формате `ДД.ММ.ГГГГ` с маской ввода;
- слайдер в личном кабинете с автопереключением каждые 3 секунды;
- карточки заявок для мобильного отображения;
- toast-уведомления и модальное подтверждение изменения статуса;
- фильтры, сортировка и пагинация в панели администратора.
