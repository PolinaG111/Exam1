# Мобильные заметки по дизайну

- таблицы в кабинете и админ-панели заменяются карточками на малых экранах;
- изображения ограничены фиксируемой высотой через `clamp(...)`;
- крупные блоки упрощены для удобства прокрутки одной рукой;
- основные действия вынесены в заметные кнопки шириной `100%` на мобильных экранах.
